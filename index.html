Newsletter

Click to view project

HTML, CSS, JavaScript, PHP and Mysql project, that represents an example of subscription to a newsletter. The page is pixel-perfect. Bootstrap or similar frameworks are not used. The page is responsive for all devices. Social icons have hover effects as per style guides. Hover effects are present on the site as per style guides. A font with custom icons is used instead of separate icon files. SASS preprocessor is used. There is an error message under input that shows validation messages. The button is disabled if the form is not valid. On successful validation, a success message appears in the place of the form, as per design. Data that is submitted is saved in a MySQL database. Data is validated also in PHP, and if JavaScript is disabled it displays errors in the same place and style as it is with JavaScript enabled. A page is created where all saved data can be seen: Sorting by name and date is added. It is possible to filter email addresses by email providers. It is possible to search for email addresses and use filters and sorting at the same time. It is possible to delete email addresses.
